uSwitch Browser tools

Suite of favelets, chrome extensions, audits and snippets to help test and debug the uSwitch website in the browser.

browser favelet: grid overlay

  • Displays the 12 column grid used by any uSwitch page, overlaid on top of the content
  • content is still usable through the grid (in webkit/chrome/safari)
  • Shows the type and version of stylesheet running (ie, because the grid layout changed in 4.0.1)
  • The overlay is responsive as the page size is changed
  • Any updates to this favelet will be inherited automatically, no re-install is needed.
  • Doesn't work in many comms pages as they use a different stylesheet system

To install in your browser, drag the below link to your bookmark bar.

> uSwitch grid <
