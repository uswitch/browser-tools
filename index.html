uSwitch Browser tools

Suite of favelets, chrome extensions, audits and snippets to help test and debug the uSwitch website in the browser.

favelet: grid overlay

Displays the 12 column grid used by any uSwitch page, overlaid on top of the content (content is still clickable through the grid in webkit). Also shows the type and version of stylesheet running (ie, because the grid changed in 4.0.1 onwards). Doesn't work in many comms pages as they use a different stylesheet system.
